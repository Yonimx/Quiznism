    • Naming Molecular Compounds
    Lesson 13
    Molecular compounds are formed when two or more non-metal atoms bond together. Naming these compounds is crucial for communication and understanding of chemical reactions. Here are the steps to follow when naming molecular compounds.

    1. Write the name of the first element in the compound.

    2. Write the name of the second element in the compound, but change the ending to "-ide"

    3. Use prefixes to indicate the number of atoms of each element in the compound. The prefix "mono-" is only used for the second element if there is only one atom of that element present.

    For example, consider the compound made up of one carbon and two oxygen atoms. The first element is carbon, and the second element is oxygen. Since there are two oxygen atoms, we use the prefix "di-". Therefore, the name of the compound is carbon dioxide.



    Here is a table of common prefixes used when naming molecular compounds:
    Prefix Number of Atoms
    mono- 1
    di- 2
    tri- 3
    tetra- 4
    penta- 5
    hexa- 6
    hepta- 7
    octa- 8
    nona- 9
    deca- 10



    Here are some examples of molecular compound names:
    Compound Formula Compound Name
    CO Carbon monoxide
    N20 Dinitrogen monoxide
    PCl5 Phosphorus pentachloride
    SF6 Sulfur hexaflouride
    H2O Dihydrogen monoxide



    In summary, naming molecular compounds involves identifying the elements present in the compound, using prefixes to indicate the number of atoms of each element, and changing the ending of the second element to "-ide".
